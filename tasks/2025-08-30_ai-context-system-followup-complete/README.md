## 🚀 사용 방법

### Context 없는 새로운 AI 세션 시작 시

#### 1단계: 필수 문서 순서대로 읽기
```
1. README.md (이 문서) - 전체 개요 파악
2. QUICK_START_GUIDE.md - 프로젝트 즉시 이해
3. AI_CONTEXT_SYSTEM_IMPLEMENTATION.md - 시스템 구조 이해
4. FOLLOWUP_TASKS_COMPLETE_ANALYSIS.md - 실행 계획 파악
```

#### 2단계: 첫 번째 작업 시작
```bash
# 권장 첫 명령어
/analyze scripts --focus quality --systematic --evidence --validate

# 목적: 현재 검증 스크립트들의 중복을 분석하고 개선 방안 제시
```

#### 3단계: Phase별 진행
- **Phase 1**: 정리 및 안정화 (1-2주)
- **Phase 2**: 성능 최적화 (3-5주) 
- **Phase 3**: 고도화 및 지속성 (6-9주)
