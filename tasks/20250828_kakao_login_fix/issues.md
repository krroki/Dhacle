## 🔍 현재까지 발견된 문제

1. **OAuth 리다이렉트 URI 불일치**
   - 카카오 개발자 콘솔에 `https://dhacle.com/auth/callback` URI 미등록 추정
   - Supabase Dashboard OAuth 설정 확인 필요

2. **환경변수 불일치**
   - `NEXT_PUBLIC_SITE_URL` 설정 확인 필요

3. **세션 만료 문제**
   - 카카오 로그인 시 즉시 `session=expired` 발생
